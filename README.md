# ThumbForge ComfyUI

YouTube thumbnails generation workflow for ComfyUI.
This workflow is optimized for 1280x720 (16:9) resolution.

## Features
- **Aspect Ratio Conversion**: Converts 3:2 landscape images (e.g., generated by ChatGPT/DALL-E) to the YouTube-recommended 16:9 (1280x720) format.
- **AI Outpainting**: Uses generative AI to naturally fill in the side margins created during conversion.
- **Gacha Element**: Since it relies on generative AI, the outpainting result may vary. It might require multiple runs (gacha) to get the perfect result.

## Prerequisites
- [ComfyUI](https://github.com/comfyanonymous/ComfyUI)

### Recommended Model
For best results with outpainting, we recommend using:
- **Juggernaut XL Inpainting**: [Download from Civitai](https://civitai.com/models/862813/juggernaut-xl-inpainting-updated)
  - Filename: `juggernautXLInpainting_xiInpainting.safetensors`

### Required Custom Nodes
- **kikotools** (Required for `ImageScaleDownBy` node)
  - Install via [ComfyUI-Manager](https://github.com/ltdrdata/ComfyUI-Manager).

## Installation & Usage
1. Install the required custom nodes and model.
2. Drag and drop the `.json` file into ComfyUI.
3. **LoadImage**: Upload your 3:2 image.
4. **CLIPTextEncode**: Enter prompts to describe the environment for outpainting.
5. **KSampler**: Generate and enjoy the "gacha" process!
